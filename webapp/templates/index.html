ebimport_splash

Ajout des inscriptions à une base SPLASH Meet Manager (.mdb) à partir d'un fichier Excel Eventbrite. La structure des épreuves (styles, événements, groupes d'âge, sessions) doit déjà exister dans la base .mdb ; l'outil ne fait qu'y ajouter les clubs, athlètes et entrées.

Fichier source
Mode d'exécution
Fichier .mdb SPLASH *

Obligatoire. La structure des épreuves (styles, événements, groupes d'âge, sessions) est prise telle quelle depuis ce fichier — le chargeur n'ajoute que les clubs, athlètes et inscriptions. Si aucune inscription n'est présente dans le .mdb, c'est considéré comme un premier chargement ; sinon, c'est une mise à jour additive.

Résultats

⚠ Erreurs fatales — importation annulée

Un ou plusieurs billets du fichier Excel n'ont pas d'équivalent dans le modèle .mdb. Le chargeur s'arrête sans rien écrire — corrigez le fichier Excel ou ajoutez les épreuves manquantes dans SPLASH, puis relancez.
